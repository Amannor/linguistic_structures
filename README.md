# linguistic_structures
This project is to search specific linguistic structures in given text\s

This project is part of an academic research done under the auspices [Prof. Anat Prior from Haifa university](https://langnum.haifa.ac.il/EPriorMembers.php)   
To run this:
1) Install the reqiured packages detailed in the requirements.txt file (can run pip install -r requirements.txt).
2) Change the value of the global variables that hold the location of the input and output dir
3) Run the programa: python main.py

The program was written and run in python 3.10
